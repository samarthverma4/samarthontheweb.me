# Assets Directory

This directory contains static assets for the portfolio website.

## Contents

- `Samarth Verma - Resume.pdf` - Resume file (accessible at `/assets/Samarth Verma - Resume.pdf`)

## Usage

Files placed in this directory are publicly accessible and served from the `/assets` URL path in production.

To update assets:
1. Add or replace files in this directory
2. Reference them in the code using `/assets/filename`
