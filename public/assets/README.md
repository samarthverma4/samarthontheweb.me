# Assets Directory

This directory contains static assets for the portfolio website.

## Contents

Place your resume and other static files here. Files in this directory are publicly accessible.

## Usage

Files placed in this directory are served from the `/assets` URL path in production.

To add assets:
1. Add files to this directory (e.g., `Samarth Verma - Resume.pdf`)
2. Reference them in the code using `/assets/filename`

## Example

For a resume file named `Samarth Verma - Resume.pdf`:
- File location: `public/assets/Samarth Verma - Resume.pdf`
- URL path: `/assets/Samarth Verma - Resume.pdf`
- Code reference: `<a href="/assets/Samarth Verma - Resume.pdf" download>`
